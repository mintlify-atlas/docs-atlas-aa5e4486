# Welcome to Backend App

Backend App is a comprehensive REST API service built with Laravel 12 that provides a complete solution for managing educational institutions, academic structures, and evaluation systems.

## What You Can Build

This API enables you to build educational platforms with:

- **Multi-institution support** — Manage multiple educational institutions with separate academic years, schedules, and contact information
- **User management** — Handle student and teacher accounts with role-based access
- **Course structure** — Organize curriculum into sections, units, and topics
- **Learning resources** — Attach documents, videos, and materials to topics
- **Assessment tools** — Create evaluations with questions and response options
- **Registration system** — Track student enrollments in different sections

## Technology Stack

- **Laravel 12** — Modern PHP framework with elegant syntax
- **Laravel Sanctum** — API token authentication
- **SQLite** — Lightweight database (supports MySQL/PostgreSQL)
- **RESTful Architecture** — Standard HTTP methods and JSON responses

## API Architecture

All API endpoints follow a consistent structure:

- **Base URL**: `/api`
- **Authentication**: Bearer token via Laravel Sanctum
- **Response Format**: Standardized JSON with `success`, `message`, and `data` fields
- **CRUD Operations**: Full Create, Read, Update, Delete support for all resources
